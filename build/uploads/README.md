# Directorio de Uploads - Boletines

Este directorio almacena los archivos adjuntos de los boletines.

## Formatos soportados:
- **Documentos**: PDF, Word (.doc, .docx), Excel (.xls, .xlsx), PowerPoint (.ppt, .pptx)
- **Imágenes**: JPG, JPEG, PNG, GIF, WebP
- **Video**: MP4, WebM, OGG
- **Audio**: MP3, WAV, OGG
- **Texto**: TXT, CSV

## Límites:
- Tamaño máximo: 50MB por archivo
- Los archivos se renombran automáticamente para evitar conflictos

## Seguridad:
- Solo se permiten tipos de archivo específicos
- Los nombres de archivo se sanitizan
- Se valida el tipo MIME
